Trata-se de um jogo de combate desenvolvido em Dart, utilizando os principais conceitos de Programação Orientada a Objetos (POO). O projeto aplica o uso de classes abstratas, herança, polimorfismo, métodos sobrescritos, funções, estruturas condicionais e laços de repetição.

O sistema simula uma batalha entre personagens distintos (como Guerreiro e Mago), cada um com habilidades específicas. O fluxo do jogo é controlado por turnos, definidos aleatoriamente, e há interação com o usuário por meio de entradas via terminal, permitindo a escolha das ações durante o combate.
